## SQL Advanced Topics Exercise

### Material
**SQL Fundamentals – Data Engineering & Analytics – Week 5–9**

This exercise covers the following topics:

- **Case Statements** – Using `CASE` for conditional logic  
- **IIF vs CASE** – Understanding which to use and when  
- **Common Table Expressions (CTEs)** – Writing readable and reusable queries  
- **Chaining Multiple CTEs** – Organizing multi-step queries  
- **Subqueries** – Nested queries for filtering, calculations, or logic  
- **Window Functions** – `ROW_NUMBER()`, `RANK()`, `SUM() OVER()`, `AVG() OVER()`  
- **UNION Operations** – Combining multiple result sets  
- **Intervals** – Working with time intervals in SQL  
- **Date & Time Arithmetic** – Adding, subtracting, and comparing dates  
- **DDL (Data Definition Language)** – Commands like `CREATE`, `ALTER`, `DROP`  
- **DML (Data Manipulation Language)** – Commands like `INSERT`, `UPDATE`, `DELETE`  
- **Data Conversions** – Converting between data types
